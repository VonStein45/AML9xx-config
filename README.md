# AML9xx-config
Small config tool for Armbian "AML9xx-box" image

Toolchains necessary to build:

MINGW (Only for my test, but also gives the smallest Windows exe): https://sourceforge.net/projects/mingw/

Not tested yet, but mingw-x64 might also work: https://www.mingw-w64.org/

ZIG: Multi target cross compiler for windows as well as Linux (x64 and ARM64): https://ziglang.org/ (https://ziglang.org/download/)
